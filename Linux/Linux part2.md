##1.1 认识计算机的硬件配备##
>- CPU :
  - CPU 速度的优良除了核数之外，判断同级产品之间的速度，就靠频率，频率是指：单位时间的运算次。
  - CPU 有[倍频] 和 [外频] 
  - 外频：是 CPU 与接口设备进行数据传输/运算的速度
  - 倍频：是 CPU 本身运算时候加上去的一个运算速度！ 两者相乘才是 CPU 的频率

>- RAM (内存)
  - 所有需要操作的数据，都需要先读入内存，才能喂给CPU进行运算。
